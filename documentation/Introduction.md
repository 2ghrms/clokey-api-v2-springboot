# 👕Clokey Docs

> 최종 수정 일자 : 8월 28일
> 
> 최종 수정자 : 나용준

- Clokey Docs는 Clokey 서버분들을 위해서 Clokey의 서비스 설계와 코드에 대한 이해를 돕고자 작성되었습니다.
- Legacy 코드를 수정해야 하는 일이 발생하지 않도록 처음부터 좋은 코드 구조를 가지고 명확하게 개발 할 수 있도록 노력해요!

### 📌 안내 사항

- 주제 또는 클래스 분류 단위로 안내 문서를 작성할 예정입니다. 
- 기존에 존재하는 컨벤션을 참고하고 수정할 부분이 있다면 Spring Boot회의를 통해서 안건을 제시하고 모두가 알 수 있도록 수정 사항을 반영해 주세요.
- 자신이 전혀 새로운 기능에 대해서 구현을 했다면, 꼭 기록을 남겨주세요.
- 모든 문서는 최종 수정 일자와 최종 수정자를 꼼꼼히 기록 부탁드립니다.

### 📌 분류 

- ```feature``` : 기능 단위 안내 사항
- ```convention``` : 컨밴션과 관련된 사항
