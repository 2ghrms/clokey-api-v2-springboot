# 📌 공통 적용 사항

> 최종 수정 일자 : 8월 28일
>
> 최종 수정자 : 나용준 

- ```주석 최소화``` : 현재 구체적인 Code Convention을 적용하고 있기 때문에 컨벤션에서 정의 되지 않았거나, 예외적인 부분들을 제외하고는 주석을 최소화 부탁드립니다.
- ```클린 코딩``` : Stream 문법 사용과 중복 최소화등 각자 코딩하실때 좋은 구조를 고민해서 작성 부탁드립니다.
- ```이름에 의미를 담자``` : 기본적인 사항이지만, 항상 고민 부탁드립니다.
- ```미사용 Import 제거 관련``` : 기본적으로 ```Spotless```가 잡아주긴 하지만 필드로 주입되는 경우와 같이 삭제되지 않는 경우도 있습니다. 
- ```공백과 코드 포메팅``` : ```Spotless```로 관리됩니다.
