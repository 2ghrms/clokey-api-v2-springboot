# 📌 Branch Convention

> 최종 수정 일자 : 8월 28일
>
> 최종 수정자 : 나용준

- ```main``` : Prod 배포 프랜치 
- ```develop``` : Dev 배포 브랜치
- ```feature/#이슈번호-설명``` : 기능 개발 브랜치
    - 생성한 이슈번호로 브랜치를 생성한다.
    - 이슈번호 뒤에 브랜치 설명을 영어로 적어둔다.
    - ex) feat/#20-login

⚠️ Squash & Merge 이후에 반드시 Branch를 즉시 삭제할 필요는 없습니다! 잠시 남겨둘 필요가 있다면 천천히 삭제해 주세요.
