# 📌 소셜 로그인과 Swagger에서 로그인

> 최종 수정 일자 : 8월 28일
>
> 최종 수정자 : 나용준

- api : Controller, Service
- common : 공통적으로 사용되는 Utility, Annotation, Validation, Exception 등등.. 모든 모듈에 공통적으로 사용될 수 있는 것들이 여기에 속합니다.
- domain : Entity , Repository interface가 여기에 속하고 말 그대로 비즈니스의 도메인을 담고 있습니다.
- batch : 주기적 작업, 통계 처리, 그리고 대규모 처리 등등
- infrastructure : 외부와의 Config 파일이 여기에 속합니다.

### ✅ Clokey 프로젝트는 Repository가 API 영역에 존재합니다

- 그 이유는 Projection을 사용하기 때문입니다. 
- Projection을 사용하면 Layer에서도 역 의존성이 발생하는데, 이는 멀티 모듈에서는 허용되지 않기 때문에 전체를 API로 옮겼습니다.
